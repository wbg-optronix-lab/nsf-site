Relax-II GeSi Simulator

TO MAKE THE PROGRAM RUN: This program runs on Windows. You will need to create a directory C:\data for the output files to be written to. You will need a file msvbvm50.dll in directory C:\Windows\SysWOW64 (available in this support document.)
